Create a function that calculates the factorial of a number and use it in your script.

def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n - 1)

Task 7: Create a Contact List

Create a contact list program using dictionaries to store names and phone numbers. Implement functionality to add, remove, and search contacts.

contacts = {}

def add_contact(name, number):
    contacts[name] = number

def remove_contact(name):
    if name in contacts:
        del contacts[name]

def search_contact(name):
    return contacts.get(name, "Contact not found")

add_contact("Alice", "123-456-7890")
add_contact("Bob", "987-654-3210")

1. Object-Oriented Programming (OOP)

Learn the principles of OOP, including classes, objects, inheritance, and polymorphism.

Task 10: Define a Class

Create a class Car with attributes for make, model, and year. Add methods to display the car's details and age.

class Car:
    def __init__(self, make, model, year):
        self.make = make
        self.model = model
        self.year = year

    def display_details(self):
        return f"{self.year} {self.make} {self.model}"

    def car_age(self):
        current_year = 2024
        return current_year - self.year

car = Car("Toyota", "Corolla", 2015)
print(car.display_details())
print("Car age:", car.car_age())

Task 11: Error Handling Example

try:
    num = int(input("Enter a number: "))
    result = 10 / num
except ZeroDivisionError:
    print("Error: Division by zero is not allowed.")
except ValueError:
    print("Error: Invalid input. Please enter a number.")
finally:
    print("Execution complete.")
